Basic Tetris
============

A Tetris clone written in C++ and using SDL2.

How to build
------------

1. Make sure you have dependencies `SDL2`
2. `git clone https://github.com/Loico/basic_tetris.git basic_tetris`
3. `cd basic_tetris`
4. `make`

Controls
--------

- Arrow left/right - Move left/right
- Arrow down - Drop faster
- Space - Rotate piece
- Escape - Pause
